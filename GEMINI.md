# Project Overview
Balance Buddy is a financial tracking Progressive Web App (PWA) designed to help users manage accounts, budgets, and transactions. It operates as a client-side application, likely using IndexedDB for local data persistence.

## Technical Stack
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Storage**: IndexedDB (Local-first, offline capable)
- **Type**: Progressive Web App (PWA)

## Key Features
- **Account Management**: Track multiple accounts.
- **Budgeting**: Set and monitor budgets.
- **Transaction Logging**: Add, edit, and list transactions.
- **Data Management**: Import/Export (JSON, CSV), Purge data.
- **Filtering**: Filter transactions by criteria.
- **Mobile-First**: Responsive design with touch-friendly controls.

## Directory Structure
- `index.html`: Main application entry point.
- `script.js`: Core application logic.
- `style.css`: Styling.
- `sw.js`: Service Worker for offline support.
- `manifest.json`: PWA configuration.
- `start_banking_server.bat`: Local dev server script.

## Current Status
- **Functional**: UI suggests a complete feature set for personal finance tracking.
